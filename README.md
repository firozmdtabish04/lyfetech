# 🚀 LYFETECH

A modern full-stack web application built with **React, Tailwind CSS, and a Spring Boot backend**, designed to deliver a smooth and responsive user experience.
---

## 📌 Features

- 🔐 User Authentication (Login / Signup)
- 👤 Profile Management
- 📦 Dynamic Data Handling via API
- 🎨 Responsive UI with Tailwind CSS
- ⚡ Fast frontend powered by Vite
- 🔗 REST API integration
- 🌐 Deployed on Vercel

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Vite
- React Router

### Backend
- Spring Boot
- REST APIs
- PostgreSQL (Supabase)

### Deployment
- Vercel (Frontend)
- Render (Backend)
- Supabase (Database)
